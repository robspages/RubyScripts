setup

Install Ruby and Rails using RailsInstaller from http://railsinstaller.org/en

In your resulting Sites folder create a new project folder 
in the new project folder run the following in cmd line: 
gem install watir
gem install nokogiri 


running the script files is done by calling them directly in cmd:

ruby Promo-Restrictions.rb

-or- using the build command in a Ruby aware editor like Sublime Text
